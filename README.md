Personopplysninger
================

Dette er frontend prosjekt for applikasjon som skal gi brukerne innsikt i all informasjon NAV har om den.

# Komme i gang

For å kjøre lokalt:

1.  `npm install`
2.  `npm run start`

For å kjøre tester:

1.  `npm install`
2.  `npm run test`

Bygge for prod:

1.  `npm install`
2.  `npm run build`
