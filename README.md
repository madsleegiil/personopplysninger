Personopplysninger
================

Dette er frontend prosjekt for applikasjon som skal gi brukerne innsikt i all informasjon NAV har om den.

# Komme i gang

### Frontend
1. `npm install`
2. `npm start`

For å kjøre tester:

`npm run test`

---

# Henvendelser
